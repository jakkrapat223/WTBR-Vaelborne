HTML → UMG JSON Converter

Loads a prototype HTML page, measures the rendered DOM, and emits UMGJsonGenerator JSON. Validate the result in the preview page before importing into Unreal.

Serve the repo root, e.g. python -m http.server 8743 from the project root, then open this page via http://localhost:8743/Plugins/UMGJsonGenerator/WebPreview/converter.html. The element treated as the 1920x1080 canvas. Use body if the page has no stage wrapper.
